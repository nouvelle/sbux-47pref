# STARBUCKS 47 JIMOTO Frappuccino 投稿サイト

# このリポジトリについて / About

このフラペチーノの画像を投稿するサイトのリポジトリです。  
実際のコードは [リポジトリ](https://github.com/nouvelle/sbux-47pref#リポジトリ) にあります。
